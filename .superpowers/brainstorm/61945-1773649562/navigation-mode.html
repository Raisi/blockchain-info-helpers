Navigation: Frei oder geführt?

Sollen die Tabs weiterhin frei anklickbar sein, oder soll der User den Weg linear durchlaufen?

A

Immer frei navigierbar

Alle Tabs sind jederzeit klickbar. Die Pipeline-Bar zeigt den empfohlenen Weg, aber der User kann springen. Tabs ohne "Input" vom vorherigen Tab nutzen Standardwerte.

Beispiel: Tab 3 direkt geöffnet → nutzt Standard-Basispunkt und Skalar 2
B

Frei, aber mit sanftem Hinweis

Alle Tabs klickbar. Wenn der User einen Tab überspringt, erscheint ein dezenter Hinweis: "Du hast Punkt-Addition übersprungen — dieser Tab baut darauf auf." mit Link zurück. Kein Blocker.

⚡ Tipp: Schau dir zuerst "Punkt-Addition" an — dieser Tab baut darauf auf. → Zur Addition
C

Linear freischalten

Tabs 2–4 sind initially ausgegraut/gesperrt. Der User muss Tab 1 "abschließen" (z.B. Punkt setzen) bevor Tab 2 verfügbar wird. Gamification-Ansatz.
